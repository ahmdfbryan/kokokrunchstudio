KokoKrunch Studio CMS v2.1

Perbaikan:
- Ringkasan pesanan sekarang memiliki heading "Metode Pembayaran".
- Metode pembayaran di halaman order dibuat dropdown agar lebih rapi.
- Detail pembayaran berubah otomatis sesuai pilihan Bank, E-Wallet, atau QRIS.
- Tombol copy nomor rekening/HP ditambahkan.
- Dashboard pembayaran dipisah berdasarkan jenis: Bank, E-Wallet, QRIS.
- Input dashboard pembayaran menyesuaikan jenis yang dipilih.
- Upload gambar QRIS tetap tersedia khusus untuk jenis QRIS.
- Responsive mobile untuk ringkasan pesanan, form dashboard, dan pembayaran diperbaiki.

Cara menjalankan:
1. Extract ZIP.
2. Masuk folder kokokrunch_studio_v21_payment_responsive.
3. Jalankan: python server.py
4. Buka: http://localhost:8000
